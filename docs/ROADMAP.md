## GitHub Project 看板模板

建议创建一个 Project（Board），列结构如下：

1. `Backlog`
2. `Ready`
3. `In Progress`
4. `In Review`
5. `Done`

每张卡片建议包含：

- 背景问题
- 目标用户
- 验收标准（Definition of Done）
- 优先级（P0/P1/P2）
- 预估工作量（S/M/L）

## Issue 标签建议

- `type:feature`
- `type:bug`
- `type:docs`
- `priority:p0`
- `priority:p1`
- `priority:p2`
- `status:blocked`
